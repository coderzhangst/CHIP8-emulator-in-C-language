## Rom resources:
Download chip8 public rom on https://www.zophar.net/pdroms/chip8/chip-8-games-pack.html

## Emulator Descriptions:
1.Run a rom on CHIP8.x86.exe or CHIP8.x64.exe (full screen)
    Usage on Windows CMD: CHIP8.x86.exe YourRomFilePATH or CHIP8.x64.exe YourRomFilePATH
2.press esc to quit chip8 emu
